# jmes-cogs
jmes cogs for Red-DiscordBot

# Installation
```
[p]repo add jmes-cogs https://github.com/jmesfo0/jmes-cogs
[p]cog install jmes-cogs pexels
[p]cog install jmes-cogs tacoshack
```
[p] = Your bot's prefix
